Website Structure:
-----------------
Single-page responsive website with the following sections:
1. Hero Section - Welcome message, business description, and 6 service cards
2. Gallery Section - Filterable image gallery of cakes
3. Reviews Section - Customer testimonials slider (5 reviews)
4. Footer - Contact information and social links

Key Features:
------------
✓ Responsive design (mobile, tablet, desktop)
✓ Smooth scrolling navigation
✓ Image zoom hover effects on service cards
✓ Auto-playing reviews carousel
✓ Filterable gallery with lightbox
✓ Professional typography (Great Vibes, Playfair Display, Lato)
✓ Brand colors: Pink (#d4145a) and earth tones

Technical Details:
-----------------
Framework: Bootstrap 3
Libraries: jQuery, Owl Carousel, Isotope (gallery filtering)
Custom CSS Files:
  - css/custom-fonts.css (typography)
  - css/hero-cards.css (service card styles with hover effects)
  - css/reviews-slider.css (reviews carousel styling)
  - css/smooth-scroll.css (smooth page scrolling)
